Este proyecto incluye un archivo XML de ejemplo y su DTD para facturas electrónicas. El XML contiene la estructura básica de una factura comercial con emisor, cliente y líneas de detalle. La DTD valida la estructura y atributos requeridos.

Para usar:

Modifica factura.xml con tus datos

Valida contra factura.dtd

Asegúrate que cumple los requisitos legales

Requisitos: Validador XML básico
